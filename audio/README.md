# Audio Files Setup

Place your Athan/Adhan audio files in this directory.

## Required Audio Files

You need audio files for the five daily prayers:

1. **fajr.mp3** - For Fajr (Dawn) prayer
2. **dhuhr.mp3** - For Dhuhr (Noon) prayer
3. **asr.mp3** - For Asr (Afternoon) prayer
4. **maghrib.mp3** - For Maghrib (Sunset) prayer
5. **isha.mp3** - For Isha (Night) prayer

## Supported Formats

- MP3 (.mp3)
- WAV (.wav)

## Where to Find Athan Audio Files

You can download free Athan audio files from:

1. **IslamicFinder**: https://www.islamicfinder.org/
2. **YouTube**: Search for "Adhan mp3" and use a YouTube to MP3 converter
3. **Islamic Audio Sites**: Many Islamic websites offer free Athan downloads
4. **Record Your Own**: You can record your local mosque's Athan

## File Naming

You can use any filename you want. The default names are:
- fajr.mp3
- dhuhr.mp3
- asr.mp3
- maghrib.mp3
- isha.mp3

But you can configure custom filenames through the web interface.

## File Size Recommendations

- Keep files under 5MB for best performance
- Typical Athan duration: 2-5 minutes
- Recommended quality: 128-192 kbps for MP3

## Testing Your Audio Files

Once you've added audio files:
1. Start the application
2. Go to http://localhost:8080
3. Use the "Test" button next to each prayer in the Audio Settings section
4. Adjust the volume as needed
